About Profiles

You need a profile to use Grain. Its Note is a name for identification which is displayed in the dropdown at the end of the navigation bar.

You will only be able to add products of the profile's currency to its inventory.

Settings will allow you to add other users to collaborate. They can do all the same things as you, including:

  • Manage the inventory
  • Create and change meals

The Consumer system allows you to create tags and associate specific meals with the people who ate them, whether or not they use Grain.Each meal must have a Consumer. If you have not done so, create a consumer associated with your username.
